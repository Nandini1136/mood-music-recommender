
# AI Mood Music Recommender 🎵

## 📌 Overview

This project recommends songs based on the user's current mood using basic AI techniques.

## 🎯 Objective

To provide personalized music suggestions based on emotional state.

## 🧠 Concepts Used

* Rule-based AI system
* Data filtering

## ⚙️ Features

* Input mood from user
* Recommend songs based on mood
* Simple and interactive

## ▶️ How to Run

```bash
python src/app.py
```

## 📊 Sample Output

Enter your mood: happy
Recommended songs:

* Shape of You
* Happy

* Basics of AI recommendation systems
* Working with datasets
